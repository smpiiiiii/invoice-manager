🤖 自動処理設定

毎日自動でメールを処理します。
下のボタンでトークンをコピーし、Vercelの環境変数 CRON_REFRESH_TOKEN に設定してください。
Vercel ダッシュボード → Settings → Environment Variables に貼り付け
📝 設定手順:
1️⃣ 上のボタンでトークンをコピー
2️⃣ Vercel Dashboard → プロジェクト → Settings
3️⃣ Environment Variables に追加:
CRON_REFRESH_TOKEN = (コピーしたトークン)
CRON_SECRET = (任意のランダム文字列)
4️⃣ 再デプロイすれば毎日18時に自動処理 🎉
