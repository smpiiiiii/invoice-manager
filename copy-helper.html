📋 GASコード コピーヘルパー

各ファイルの「コピー」ボタンを押して → GASエディタの該当ファイルで Ctrl+A → Delete → Ctrl+V で貼り付け → Ctrl+S で保存

① コード.gs

GASエディタで「コード.gs」をクリック → Ctrl+A → Delete → 下のコピーボタン → Ctrl+V → Ctrl+S

② 設定.gs

GASエディタで「設定.gs」をクリック → Ctrl+A → Delete → 下のコピーボタン → Ctrl+V → Ctrl+S

③ ダッシュボード.html

GASエディタで「ダッシュボード.html」をクリック → Ctrl+A → Delete → 下のコピーボタン → Ctrl+V → Ctrl+S

④ ダッシュボード機能.gs

GASエディタで「ダッシュボード機能.gs」をクリック → Ctrl+A → Delete → 下のコピーボタン → Ctrl+V → Ctrl+S

✅ 全部貼り付けたら：
1. 「デプロイ」→「新しいデプロイ」
2. 歯車マーク → 「ウェブアプリ」
3. 実行: 「自分」/ アクセス: 「全員」
4. 「デプロイ」ボタン → URLが表示される！
